About the Project

The Challenge

The Pacific Dataviz Challenge highlights a critical reality: Pacific Island countries are uniquely vulnerable to the impacts of climate change. From rising sea surface temperatures disrupting local marine ecosystems to shifting weather patterns threatening agricultural yields, the consequences are multi-faceted.

The primary obstacle for policymakers and stakeholders is making sense of these massive, disconnected datasets. There is an urgent need to translate abstract data into tangible, localized narratives that clearly illustrate vulnerabilities and pathways to resilience.

The Application

This web application is built to answer that challenge. By leveraging a Python backend (FastAPI) and dynamic frontend visualizations (Plotly), the app bridges the gap between raw data and public comprehension.

We employ a Progressive Disclosure framework. Rather than overwhelming the user with a dashboard of 7 disconnected charts, metrics are grouped logically into thematic chapters. Furthermore, regional benchmarking is integrated directly into the charts, allowing users to instantly compare a country's localized crisis against the broader Pacific context.

Datasets Utilized

This project harmonizes multiple distinct datasets to create a unified narrative:

Ocean & Atmospheric Health

  • Mean sea surface temperature anomalies
  • Meteorological monitoring network (disaggregated)

Agro-Ecosystems & Food Security

  • Crop yield (disaggregated)
  • Livestock yield (disaggregated)

Socio-Economics & Infrastructure

  • Tourist arrivals (disaggregated)
  • Power generation (disaggregated)
  • Environmental taxes (disaggregated)
