About the Project

The Challenge

The Pacific Dataviz Challenge highlights a critical reality: Pacific Island countries are uniquely vulnerable to the impacts of climate change. From rising sea surface temperatures disrupting local marine ecosystems to shifting weather patterns threatening agricultural yields, the consequences are multi-faceted.

One primary obstacle for policymakers and stakeholders is making sense of open source data to better understand the impact. There is an urgent need to translate abstract data into tangible, localized narratives that clearly illustrate vulnerabilities and pathways to resilience.

The Application

This web application is built to answer that challenge. By leveraging a clear narrative and data visualization, the app bridges the gap between raw data and public comprehension. It is built using fastAPI, Pandas, Plotly and Vercel.
